# login-signup-system
Features: <br>
	1. SignUp system <br>
	2. Login system <br>
	3. Users can login with both username and email <br>
	4. Preventing user from login if entered Username/Email/Password is incorrect <br>
	5. Preventing user from accessing "dashboard.php" without login <br>
	6. Preventing user from signup if entered "Username/Email/Mobile" already exists <br>
	7. Users can change their Firstname, Lastname, Mobile, Email, Username and Password <br>
	8. Preventing user from changing "Username/Email/Mobile" if the entered info already exists
